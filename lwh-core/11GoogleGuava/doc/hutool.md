# hutool

**一个Java基础工具类，类似于jodd和Apache commons lang的Java工具类。**

**功能**

一个Java基础工具类，对文件、流、加密解密、转码、正则、线程、XML等JDK方法进行封装，组成各种Util工具类，同时提供以下组件：

- 布隆过滤
- 缓存
- 克隆接口
- 类型转换
- 日期处理
- 数据库ORM（基于ActiveRecord思想）
- 基于DFA有限自动机的多个关键字查找
- HTTP客户端
- IO和文件
- 有用的一些数据结构
- 日志
- 反射代理类的简化（AOP切面实现）
- Setting（一种扩展Properties的配置文件）
- System（JVM和系统信息等）
- WatchService的封装（文件变动监控）
- XXXUtil各种有用的工具类
