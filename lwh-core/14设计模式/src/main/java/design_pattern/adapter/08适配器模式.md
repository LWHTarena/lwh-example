# 08适配器模式 #
归属 结构型模式

=====================================================

三大范畴:

	结构型模式： 用于处理类或对象的组合
	行为型模式： 用于描述对类或对象的怎样交互和怎样分配职责
	创建型模式：主要用于创建对象

====================================================

适配器模式 --常用

+ 类适配器

+ 对象适配器

+ 接口适配器

**特点:** 将一个接口装换成客户端的一个接口

**关系:** 适配器接口和被适配器接口没有层次关系

**举例:** Java AWT 、 Java I/O 、Spring Web MVC
