# 10建造者模式 #

归属 创建型模式

=====================================================

三大范畴:

	结构型模式： 用于处理类或对象的组合
	行为型模式： 用于描述对类或对象的怎样交互和怎样分配职责
	创建型模式：主要用于创建对象

====================================================


建造者模式：将一个复杂对象的构建与它的表示分离， 使得同样的构建过程可以创建不同的表示。

一定包含4个角色

- product
- Builder抽象建造者
- ConcreteBuilder具体建造者
- Director导演类
